# 🚀 Trello Backend Clone

A scalable Trello-like backend built using Node.js, Express, MongoDB, and JWT.  
This project demonstrates multi-organization support, board management, and task lifecycle handling.

---

## 🧩 Features

- 🔐 Authentication (Signup / Signin with JWT)
- 🏢 Organization Management
- 👥 Add / Remove Members
- 📋 Board Creation
- ✅ Task Management (Create, Update, Delete)
- 🔄 Task Lifecycle (Todo → In Progress → Done)

---

## 🏗️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (Authentication)
